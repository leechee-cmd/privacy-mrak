# Privacy Mark - 本地处理图片水印工具

这是一个纯前端实现的图片水印工具，**100% 在你的浏览器本地运行**。

工具的设计初衷是为了让用户能在不泄露任何隐私的情况下，为证件照、票据或其他敏感图片快速添加自定义的保护水印。整个过程没有后端服务器参与，所有图片处理（读取、加水印、导出压缩）都在用户本地设备的浏览器上完成，绝不上传任何图片。

## ✨ 核心特性

- **🔒 绝对隐私安全**：100% 纯前端处理，无需网络请求上传图片，彻底杜绝隐私泄露风险。
- **🎨 现代原生设计体验**：基于 Apple macOS/iOS 风格，提供原生应用级别的高级交互和 UI 质感（暗色模式自适应）。
- **🛠️ 高度定制化**：
  - 支持拖拽添加图片。
  - 自定义水印文字，内置常见使用场景快捷模板。
  - 支持调节水印字体大小、颜色、不透明度、倾斜角度及间距分布。
  - 多类开源字体选择，下拉菜单所见即所得。
- **💾 高效导出**：实时预览添加效果，支持导出为 PNG（最高画质）、JPG（适合控制文件体积）、WebP（现代最优选择）格式。

## 🚀 如何运行

因为本项目是纯前端静态单页应用（HTML + CSS + 原生 JS），没有任何后端依赖，不需要执行任何复杂的 build 编译命令。

你可以选择下面最简单的方式在本地启动它：

1. **进入项目目录**：
   在终端进入包含 `index.html` 的目录下。

2. **一行命令启动（推荐）**：
   如果你安装了 [Node.js](https://nodejs.org/)，可以直接使用 `npx` 启动一个轻量级的本地静态服务器：
   ```bash
   npx -y serve . -l 3456
   ```

3. **访问本地地址**：
   启动成功后，打开浏览器访问：[http://localhost:3456](http://localhost:3456) 即可开始使用。

## ☁️ 如何部署到线上？

非常简单！因为没有后端服务，它是一组没有任何打包依赖的静态文件（HTML/CSS/JS）。你可以把它部署到任何支持静态页面托管的平台上。

以 **Cloudflare Pages** 为例：
1. 注册并登录 [Cloudflare Dashboard](https://dash.cloudflare.com/)。
2. 在左侧面板导航到 `Websites` -> `Pages`。
3. 点击 `Create a project`（创建项目）。
4. 选择 **直接上传资产文件夹 (Direct Upload)**。
5. 将当前项目打包成 ZIP（或者直接拖拽上传整个项目文件夹）。
6. 上传完成后，Cloudflare Pages 就会为你分配一个免费且极速的 `xxx.pages.dev` 域名，这样你随时可以用手机或电脑在线访问使用啦！

### 或者使用 Github Pages（自动化方案）：
1. 在 Github 创建一个仓库并把代码推送上去。
2. 在该项目内的 `Settings` > `Pages`，选择通过 `main` 或 `master` 分支的 `/(root)` 目录构建页面。
3. 保存后过一两分钟，Github 就会生成一个免费网页。

## 📁 目录结构

- `index.html` - 主网页骨架（入口）
- `css/style.css` - UI 样式表
- `js/watermark.js` - 水印核心绘制引擎（使用 Canvas API）
- `js/ui.js` - 控制面板、组件状态、事件绑定逻辑
- `js/app.js` - 项目初始化入口脚本
